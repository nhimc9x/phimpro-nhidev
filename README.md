## Decription

PhimPro is a movie watching website that sources its content from the `KKPhim API`, constantly updating with the latest and most frequent vietsub movies.

## Built With

This project was built using these technologies.

- `Vite`
- `React`
- `Typescript`
- `TailwindCSS`
- `React-router-dom`
- `RTK Query`
- `Clsx`
- `Swiper`
- `VsCode`
- `Vercel`
